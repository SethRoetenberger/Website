Lightsaber

There have been many different lightsabers that I have created, with each one becoming better and better than the last. I started off with 12-volt LED strips of a certain color and added a power source with a switch. My second version had an RGB LED strip with a separate remote outside the hilt to control the color of the lightsaber.

My third attempt was to create a double-sided lightsaber. I did this by using plumbing parts that would screw together as well as work separately.

I am currently working on version four which is controlled with a micro-controller and has color cycling as well as sound and an accelerometer to trigger the transitions.
