Resin

Much of the resin work that I have done has been a lesson in mold making and complex 3D modeling. I often create deep resin pours which require a leak proof mold and mold release. I originally struggled with leaking when it came to making these molds because I was using acrylic sheets that were held together using hot glue. This ended up being a bad idea because resin is exothermic and generated enough heat to melt the hot glue which ruined the mold. After this, I switched to using melamine wood and screwed it together. I then sealed the edges using silicone. Each one of my molds that have used this new technique have all worked flawlessly. Another way of using resin that I have enjoyed is creating an ocean landscape. To do this I made multiple layers which included different colored resigns that then mixed together to mimic ocean waves.
